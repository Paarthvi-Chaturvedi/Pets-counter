🐾 Pets Counter

A simple web app to count the total number of pets in a household.

Features
Increment and decrement the pet count
Displays a running total
Clean, minimal UI with no page reloads


Built With
HTML
CSS
JavaScript (Vanilla)


Getting Started

No installation or dependencies needed.


Clone the repository
bash   git clone https://github.com/your-username/pets-counter.git


Open index.html in your browser

That's it!

How to Use
Click Add to count a new pet
Click Remove to decrease the count
The total updates instantly on the screen
